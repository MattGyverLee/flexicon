flexicon API documentation¶

flexicon is a Python library for accessing FieldWorks Language Explorer (FLEx) projects via the SIL Language and Culture Model (LCM) API. It provides CRUD operations for FLEx data across the Grammar, Lexicon, Texts & Words, Notebook, Lists, and System domains.

The main entry point is the flexicon.FLExProject class. The full per-module API reference is generated below.
