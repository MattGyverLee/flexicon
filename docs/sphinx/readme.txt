The Sphinx configuration documents the ENTIRE flexicon package.

Layout:
    index.rst        - master document; toctree points at api/modules
    api/             - per-module .rst stubs, GENERATED by sphinx-apidoc
                       (not committed; regenerated on every build)
    conf.py          - Sphinx config. Notable settings:
                         * autodoc_mock_imports = clr / System / SIL, so the
                           build works without FieldWorks installed
                         * autodoc_default_options documents all members
                         * napoleon parses the Google-style docstrings

The documentation is built with:
    make build
which runs:
    sphinx-apidoc -f -o docs/sphinx/api flexicon flexicon/tests flexicon/examples
    sphinx-build docs/sphinx flexicon/docs/flexiconAPI

Output HTML lands in flexicon/docs/flexiconAPI/ (open index.html).
